# Steal This Skill — A Gentle Introduction to Agent Skills

Static HTML slide deck for the Flexion AI Guild talk (July 23). Live: https://fpigeonjr.github.io/ai-guild-talk/

## Present

```bash
open index.html
```

Navigation:

| Key | Action |
|---|---|
| `→` / `space` / click-right | Next slide |
| `←` / click-left | Previous slide |
| `f` | Toggle fullscreen |
| `Home` / `End` | Jump to first / last |

Slide position is stored in the URL hash, so refresh keeps your place.

## Structure

- `index.html` — the deck (self-contained; inline CSS/JS, no dependencies)
- `assets/` — gifs and images referenced by the deck
- `TALK-NOTES.md` — the talk's decision record: goal, spine, why the structure is what it is
